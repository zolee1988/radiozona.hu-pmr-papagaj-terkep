# PMR Papagáj térkép

Ez a projekt egy egyszerű, gyors és mobilbarát webes térkép, amely a magyarországi PMR papagájok helyét és alapadatait jeleníti meg.

A papagájok védelme érdekében nem pontos QTH van megadva, hanem csak egy hozzávetőleges pozició, 6 karakter pontossággal.

A papagájok listája a radiozona.hu Facebook oldaláról származik:  
https://www.facebook.com/radiozona.hu

## Funkciók

- Magyarországi PMR papagájok megjelenítése térképen  
- Nagyított, jól látható ikonok  
- Mobilbarát megjelenés  
- Olvasható, nagy betűs popupok  
- Felső információs fejléc (logó + cím + forrás + frissítési idő)  
- Egyszerűen bővíthető adatstruktúra a későbbi frissítésekhez
